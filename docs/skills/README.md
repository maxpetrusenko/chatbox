# Living Docs Skills

Project-specific skill specs for the Chatbox living documentation runtime.

These specs exist to keep two synchronized outputs alive:

- human docs for skim and decision-making
- AI docs for execution, refresh, and drift control

Core coverage target:

- product
- competitors
- architecture
- design
- UI
- intent
- research
- user personas

## Initial skill set

- `presearch.md`
- `brainlift.md`
- `visual-plan.md`
- `implementation-contract.md`
- `drift-review.md`

These are docs-first specs now. Later they can become reusable agent skill packages.
