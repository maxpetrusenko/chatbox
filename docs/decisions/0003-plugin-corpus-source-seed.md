# ADR 0003 — Legacy Plugin Docs Stay Source Seeds Initially

Date: 2026-03-31
Status: Accepted
Human skim: `docs/decisions/index.html#adr-0003`

## Decision

Do not mark old plugin docs as superseded yet. Mark them as source seeds in the registry first.

## Why

The new canonical plugin chain is still fresh. The old docs still contain high-value original thinking.

## Consequences

- discovery stays intact
- migration stays reversible
- later, once canonical chain is mature, superseded labels can be considered
