Repo map. source inventory. evidence grading. open questions.

Brainlift

DOK ladder. themes. risks. SPOVs. anti-goals. trust boundary.

Plan

Visual build map. phase order. checklist. acceptance criteria.

Design

UX flows. state model. auth states. edge cases. quality bar.

Implement

Bounded work packets. file touch map. no-go constraints.

Verify

Proof ledger. verified vs inferred. screenshots. tests. gaps.

7

Drift

Mismatch scan. stale propagation. repair packet. republish loop.
